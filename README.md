# Notes App

This project is a simple note-taking application built with React and Redux Toolkit. It allows users to create, edit, delete notes, and manage the note list with Redux.

## Features

- Manage note data with Redux Toolkit
- Use `moment` for date handling
- Built with Create React App
- Standard React scripts for development and production

## Technologies

- React 19
- Redux Toolkit
- React Redux
- Create React App
- Moment

## Installation

1. Navigate to the project folder:

```bash
cd notes-app
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm start
```

Open `http://localhost:3000` in your browser to view the application.

## Available Scripts

- `npm start` — Runs the app in development mode.
- `npm run build` — Builds the app for production.
- `npm test` — Runs tests.
- `npm run eject` — Exposes Create React App configuration (irreversible).

## Project Structure

- `src/App.js` — Main application component
- `src/redux/noteSlice.js` — Note state management
- `src/redux/store.js` — Redux store configuration
- `src/index.js` — Application entry point

## Notes

- This project is suitable for quickly building a note-taking app with React.
- State management is simplified with Redux Toolkit.
